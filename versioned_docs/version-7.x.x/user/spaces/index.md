# Working in teams with Spaces

Spaces are shared work areas in OpenCloud. They help teams organize content, manage access, and keep project or team files separate from personal storage.

  Adjust the storage quota available for a Space.

- [Disable and Enable a Space](./enable-disable.md)  
  Temporarily disable a Space and enable it again later when needed.

  Permanently remove a Space when it is no longer needed.

  Follow recommended structures and naming conventions for Spaces.
