# Desktop Client for Linux

The OpenCloud Desktop Client for Linux allows you to synchronize files between your Linux system and your OpenCloud instance. This section guides you through the complete lifecycle of the Desktop Client, from installation and initial setup to synchronization settings, application preferences, and upgrading to newer versions.

## In this section

- [Installation](./installation.md)  
  Install the OpenCloud Desktop Client on Linux using AppImage and AppImageLauncher.

- [Set up OpenCloud Desktop](./set-up.md)  
  Connect your OpenCloud account and complete the initial client configuration.

- [Synchronization Settings on Linux](./sync-settings.md)  
  Configure synchronization behavior, choose which folders to sync, and manage existing sync connections.

- [Settings in Linux](./settings.md)  
  Customize the Desktop Client by configuring general, advanced, network, and bandwidth settings.

- [Upgrade Desktop Client on Linux](./upgrade.md)  
  Learn how to upgrade the OpenCloud Desktop Client while preserving your existing configuration and synchronized files.
