# Overview

The OpenCloud Android App provides an intuitive interface that makes it easy to access your files, manage your account, and navigate between the app's main features. This section introduces the main interface elements and explains how to use them efficiently during everyday work.

## In this section

- [Account Menu](./account-overview.md)  
  Learn about the Account Menu, access account settings, view storage usage, and find helpful resources such as the documentation and privacy policy.

- [Files List Menu](./account-fileslist.md)  
  Explore the main files view, learn about the navigation elements, sorting and view options, and quickly access personal files, shares, spaces, uploads, and offline content.
