# Manage Files and Folders

Learn how to organize, manage, and review files and folders in OpenCloud. This section covers common day-to-day tasks such as creating and moving content, uploading and downloading files, restoring deleted items, searching for files, organizing content with tags and shortcuts, and reviewing file history and activity.

## In this section

### Organize and manage content

- [Create, rename, and move files and folders](./create-rename-move.md)  
  Create new files and folders, rename existing content, and move items to a different location.

- [Upload, download, and unzip content](./upload-download-unzip.md)  
  Upload files and folders to OpenCloud, download them to your device, and work with ZIP archives.

- [Delete and restore items](./delete-restore.md)  
  Remove files and folders and restore them later from the Deleted files area if needed.

### Find and organize content

- [Search for files and content](./search.md)  
  Find files by name or search within their content using the built-in search.

- [Add and remove tags](./tags.md)  
  Organize files and folders with tags to make them easier to find and categorize.

- [Create shortcuts to files or web links](./shortcut.md)  
  Create shortcuts to frequently used files or external websites for quick access.

### Review file history and activity

- [View file versions](./versions.md)  
  Review earlier versions of a file and restore or download them when needed.

- [Inspect file activity](./activity.md)  
  See which actions were performed on a file or folder and track recent activity.
