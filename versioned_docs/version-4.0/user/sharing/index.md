# Share Files and Folders

This section explains how sharing works in OpenCloud. It covers internal sharing with users and groups, public sharing links for external access, and File Drop links for receiving file uploads.

## In this section

- [Internal Sharing](./internal.md)  
  Share files and folders with users or groups within your organization.

- [External Sharing](./external.md)  
  Create public sharing links and configure options such as permissions, passwords, and expiration dates.

- [File Drop](./file-drop.md)  
  Allow external users to upload files without giving them access to existing content.
