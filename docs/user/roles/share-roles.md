| :----------       | :-:   | :-:      | :-:        | :-:   | :-:   | :-:       | :-:                                    |
| can view (secure) |   -   |     -    | -          | -     | -     | -         | x                                      |
| can view          |   x   |     x    | -          | -     | -     | -         | -                                      |
| can upload        |   x   |     x    | x          | -     | -     | -         | -                                      |
| can edit          |   x   |     x    | x          | x     | x     | x         | -                                      |
