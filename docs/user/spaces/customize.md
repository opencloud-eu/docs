# Customize a Space in OpenCloud

You can customize a Space either in the Space overview or inside the Space itself.

## Edit description

Click "Edit description". Enter your description in the text editor that opens, then save and close it.

<img src={require("./img/customize/description-edit.png").default} alt="Edit description" width="1920"/>
<img src={require("./img/customize/description-change.png").default} alt="Change the description" width="1920"/>

The description can also be edited directly in the Space by clicking the pencil next to it.

## Edit subtitle

Click "Edit subtitle". Enter the desired subtitle in the text editor and save it.

<img src={require("./img/customize/subtitle-edit.png").default} alt="Edit subtitle" width="1920"/>
<img src={require("./img/customize/subtitle-change.png").default} alt="Change subtitle" width="1920"/>

## Set image

Click "Edit image" and then choose "Set image". In the window that opens, select the desired image for your Space and confirm. Then select the desired crop area and confirm.

<img src={require("./img/customize/image-edit.png").default} alt="Select space image" width="1920"/>
<img src={require("./img/customize/image-size.png").default} alt="Crop space image" width="1920"/>
<img src={require("./img/customize/image-set.png").default} alt="Space image" width="1920"/>

## Set icon

Click "Edit image" and then choose "Set icon". In the window that opens, select the desired icon for your Space.

<img src={require("./img/customize/icon-set.png").default} alt="Click on set icon" width="1920"/>
<img src={require("./img/customize/icon-select.png").default} alt="Select the icon" width="500"/>
<img src={require("./img/customize/icon-spaceimage.png").default} alt="Icon as space image" width="1920"/>

Now you know how to customize a Space.
