# Shortcut

With OpenCloud you can create shortcuts to websites or files

## Create shortcut

- Open the “+ New” menu and select “Shortcut”.

  <img src={require("./img/link/shortcut-button.png").default} alt="Select Shortcut" width="1920"/>

- Enter a URL or file name.
  - You can also pick a file from the list after typing a few letters.

  <img src={require("./img/link/pop-up-window.png").default} alt="Enter link or file in popup window" width="500"/>

- Enter a name for the shortcut.

  <img src={require("./img/link/url-and-title.png").default} alt="Enter a name for the shortcut" width="500"/>

- Create the shortcut by clicking on "Create"

  <img src={require("./img/link/create-button.png").default} alt="Click on create" width="500"/>

## Use shortcut

- Open the context menü, click on "Open with" and choose “Open shortcut”.
  <img src={require("./img/link/shortcut-right-click.png").default} alt="Right click on shortcut" width="1920"/>
