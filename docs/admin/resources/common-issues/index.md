# Common Issues & Help

Use this page to find quick fixes for the most common admin issues.

## Startup and permissions

### Check whether the containers are running

Make sure the OpenCloud containers are up and running before looking elsewhere.

See [Check whether the containers are running](./containers-running.md).

### Docker Permission Issues

Fix permission errors caused by mounted directory ownership.

See [Docker Permission Issues](./docker-permissions.md).

## Browser and certificates

### Accept Self-Signed Certificates

Confirm the browser warning when using a self-signed local setup.

See [Accept Self-Signed Certificates](./self-signed-certificates.md).

## Access and account recovery

### Change Admin Password Set in `.env`

Reset or change the admin password when the `.env` value is no longer enough.

See [Change Admin Password Set in `.env`](./admin-password-reset.md).

## IDM and certificates

### Internal LibreIDM cert expires

Renew the internal IDM certificate when LDAP starts failing.

See [Internal LibreIDM cert expires](./libreidm-cert-expiry.md).
