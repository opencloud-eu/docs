# OpenCloud Desktop Client unter macOS aktualisieren

Der OpenCloud Desktop Client für macOS kann mit dem offiziellen signierten `.pkg`-Installationsprogramm aktualisiert werden.

Beim Upgrade wird die vorhandene Anwendung ersetzt, während Ihre Kontoeinstellungen, Synchronisierungskonfiguration, Präferenzen, zwischengespeicherten Daten und Zugangsdaten erhalten bleiben.

## Neuestes Installationsprogramm herunterladen

Laden Sie das neueste `.pkg`-Installationsprogramm von der offiziellen Release-Seite herunter:

- [OpenCloud Desktop Releases auf GitHub](https://github.com/opencloud-eu/desktop/releases)

Wählen Sie das Installationsprogramm passend zur Architektur Ihres Macs:

- Macs mit Apple Silicon (M1, M2, M3): `arm64`
- Macs mit Intel-Prozessor: `x86_64`

Speichern Sie das Installationsprogramm in Ihrem Download-Ordner oder einem anderen leicht auffindbaren Ort.

## OpenCloud Desktop schließen

Schließen Sie vor Beginn des Upgrades den laufenden Desktop Client:

1. Klicken Sie auf das OpenCloud Desktop-Symbol in der macOS-Menüleiste.
2. Öffnen Sie das Menü.
3. Wählen Sie OpenCloud Desktop beenden.

So wird sichergestellt, dass alle Synchronisierungsprozesse vor dem Upgrade sauber gestoppt werden.

## Installationsprogramm ausführen

So aktualisieren Sie OpenCloud Desktop:

1. Doppelklicken Sie auf die heruntergeladene `.pkg`-Datei.
2. Folgen Sie den Anweisungen im macOS-Installationsprogramm.
3. Bestätigen Sie bei Aufforderung, dass die vorhandene Installation ersetzt werden soll.

Das Installationsprogramm aktualisiert die Anwendung im Programme-Ordner.

Bestehende Konten, Synchronisierungsordner, Präferenzen, zwischengespeicherte Daten und Zugangsdaten bleiben automatisch erhalten.

## Aktualisierte Anwendung starten

Starten Sie nach Abschluss der Installation OpenCloud Desktop über einen der folgenden Orte:

- Programme-Ordner
- Launchpad
- Spotlight

Der Client sollte normal starten und Ihre vorhandene Konfiguration verwenden.

## Installierte Version überprüfen

So überprüfen Sie, ob das Upgrade erfolgreich war:

1. Öffnen Sie OpenCloud Desktop.
2. Öffnen Sie die Einstellungen.
3. Wählen Sie Info und überprüfen Sie die Version.

Die aktuell installierte Versionsnummer wird im Info-Dialog angezeigt.

:::note
Durch das Upgrade werden keine synchronisierten Dateien entfernt.

Bestehende Benutzerkonten und Synchronisierungsverbindungen bleiben konfiguriert.

Ein Neustart des Systems ist nach dem Upgrade in der Regel nicht erforderlich.
:::
