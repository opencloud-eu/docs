# OpenCloud Desktop Client für macOS

Der OpenCloud Desktop Client für macOS ermöglicht die Synchronisierung von Dateien zwischen Ihrem Mac und Ihrer OpenCloud-Instanz. Dieser Abschnitt führt Sie durch alle wichtigen Schritte – von der Installation und Ersteinrichtung über die Konfiguration der Synchronisierung und der Anwendungseinstellungen bis hin zum Upgrade auf neuere Versionen.

## In diesem Abschnitt

- [Installation](./installation.md)  
  Installieren Sie den OpenCloud Desktop Client unter macOS mithilfe des offiziellen `.pkg`-Installationsprogramms.

- [OpenCloud Desktop einrichten](./set-up.md)  
  Verbinden Sie Ihr OpenCloud-Konto und führen Sie die Ersteinrichtung des Desktop Clients durch.

- [Synchronisierungseinstellungen unter macOS](./sync-settings.md)  
  Konfigurieren Sie das Synchronisierungsverhalten, wählen Sie aus, welche Ordner synchronisiert werden sollen, und verwalten Sie bestehende Synchronisierungsverbindungen.

- [Einstellungen unter macOS](./settings.md)  
  Passen Sie den Desktop Client an, indem Sie allgemeine, erweiterte, Netzwerk- und Bandbreiteneinstellungen konfigurieren.

- [OpenCloud Desktop Client unter macOS aktualisieren](./upgrade.md)  
  Erfahren Sie, wie Sie den OpenCloud Desktop Client auf eine neuere Version aktualisieren, ohne Ihre vorhandene Konfiguration oder bereits synchronisierte Dateien zu verlieren.
