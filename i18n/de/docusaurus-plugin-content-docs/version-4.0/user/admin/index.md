# Administration von OpenCloud

In diesem Abschnitt werden die Administrationsfunktionen von OpenCloud erläutert. Abhängig von Ihren Berechtigungen können Sie Kontoeinstellungen verwalten, Benutzer, Gruppen und Spaces administrieren, App-Tokens erstellen und externe Anwendungen wie WebDAV-Clients mit OpenCloud verbinden.

## In diesem Abschnitt

- [Einstellungen](./settings.md)  
  Lernen Sie den Administrationsbereich kennen und verwalten Sie – abhängig von Ihren Berechtigungen – Benutzer, Gruppen und Spaces.

- [App-Tokens](./app-tokens.md)  
  Erstellen und verwalten Sie sichere Tokens für Drittanbieteranwendungen und WebDAV-Clients.

- [WebDAV](./web-dav.md)  
  Verbinden Sie OpenCloud-Dateien und Spaces über WebDAV mit Ihrem Betriebssystem oder Dateimanager.
