# Rollen

OpenCloud verwendet verschiedene Rollenmodelle für unterschiedliche Bereiche des Systems. Benutzerrrollen, Space-Rollen und Freigaberollen sind voneinander unabhängig und steuern jeweils unterschiedliche Berechtigungen.

Benutzerrollen legen fest, welche Aktionen ein Benutzer systemweit ausführen darf. Sie steuern administrative Berechtigungen, beispielsweise die Verwaltung von Benutzern, Gruppen und Spaces.

Space-Rollen bestimmen, welche Berechtigungen ein Mitglied innerhalb eines bestimmten Spaces besitzt. Sie regeln den Zugriff auf Inhalte sowie die Verwaltung des jeweiligen Spaces.

Freigaberollen legen fest, welche Aktionen Empfänger mit freigegebenen Dateien und Ordnern ausführen dürfen. Sie gelten ausschließlich für Freigaben und ändern weder die Benutzerrolle in OpenCloud noch die Rolle innerhalb eines Spaces.
