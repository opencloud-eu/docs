# Dateien und Ordner freigeben

In diesem Abschnitt erfahren Sie, wie Freigaben in OpenCloud funktionieren. Er behandelt interne Freigaben für Benutzer und Gruppen, öffentliche Freigabelinks für den externen Zugriff sowie File Drop-Links zum Empfangen von Datei-Uploads.

## In diesem Abschnitt

- [Interne Freigaben](./internal.md)  
  Geben Sie Dateien und Ordner für Benutzer oder Gruppen innerhalb Ihrer Organisation frei.

- [Externe Freigaben](./external.md)  
  Erstellen Sie öffentliche Freigabelinks und konfigurieren Sie Optionen wie Berechtigungen, Kennwörter und Ablaufdaten.

- [File Drop](./file-drop.md)  
  Ermöglichen Sie externen Benutzern das Hochladen von Dateien, ohne ihnen Zugriff auf vorhandene Inhalte zu gewähren.
