# Web Benutzeroberfläche und Navigation

Die OpenCloud Web Benutzeroberfläche ermöglicht den vollständigen Zugriff auf Ihre Dateien, Spaces und Freigaben direkt im Browser. In diesem Abschnitt lernen Sie die wichtigsten Bereiche der Benutzeroberfläche kennen und erfahren, wie Sie die Dateiansicht anpassen sowie Ihr Benutzerkonto verwalten.

## In diesem Abschnitt

- [Überblick](./overview.md)  
  Lernen Sie die wichtigsten Bereiche der OpenCloud Web Benutzeroberfläche sowie deren Funktionen kennen.

- [Kontoeinstellungen](./account-settings.md)  
  Verwalten Sie Ihr Benutzerkonto, passen Sie persönliche Einstellungen an und konfigurieren Sie Funktionen wie Sprache, Design oder App-Tokens.

- [Anpassungen der Datei-Liste](./file-list-adjustments.md)  
  Passen Sie die Anzeige der Dateiliste an, beispielsweise durch das Einblenden versteckter Dateien, Dateierweiterungen oder die Änderung der Kachelgröße.

- [Anzeigemodi](./view-modes.md)  
  Wechseln Sie zwischen Kachelansicht, Standard-Tabellenansicht und verdichteter Tabellenansicht, um Dateien und Ordner optimal darzustellen.
