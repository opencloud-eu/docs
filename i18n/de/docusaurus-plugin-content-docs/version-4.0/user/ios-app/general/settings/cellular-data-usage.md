# Nutzung mobiler Daten

Mit den Einstellungen **Nutzung mobiler Daten** können Sie steuern, ob OpenCloud Ihre mobile Datenverbindung verwenden darf und welche Funktionen Daten übertragen dürfen, wenn Sie nicht mit einem WLAN verbunden sind.

## Nutzung mobiler Daten konfigurieren

1. Öffnen Sie die **OpenCloud**-App.
2. Tippen Sie auf **Einstellungen**.
3. Wählen Sie **Nutzung mobiler Daten** aus.

Auf der Seite **Nutzung mobiler Daten** können Sie festlegen, wie die App Ihre mobile Datenverbindung verwendet.

<img src={require("../../img/settings/cellular-data-usage/cellular data usage.png").default} alt="Schaltfläche Nutzung mobiler Daten" height="650"/>

## Allgemeine Einstellungen

Unter **Allgemein** steht folgende Option zur Verfügung:

- **Mobilen Datenzugriff erlauben** – Aktiviert oder deaktiviert die Nutzung mobiler Daten für die gesamte OpenCloud-App.

Wenn diese Option deaktiviert ist, überträgt OpenCloud Daten nur über WLAN.

## Funktionsspezifische Einstellungen

Sie können unabhängig voneinander festlegen, welche Funktionen mobile Daten verwenden dürfen.

Die folgenden Optionen stehen zur Verfügung:

- **Offline verfügbar** – Erlaubt das Herunterladen und Synchronisieren von Dateien, die als offline verfügbar markiert sind, über eine mobile Datenverbindung.
- **Foto-Upload** – Erlaubt Auto Upload, Fotos über mobile Daten hochzuladen.
- **Video-Upload** – Erlaubt Auto Upload, Videos über mobile Daten hochzuladen.

## Funktionsweise der mobilen Datennutzung

:::important

Das Deaktivieren von **Mobilen Datenzugriff erlauben** überschreibt alle funktionsspezifischen Einstellungen.

Wenn diese Option deaktiviert ist, verwendet OpenCloud ausschließlich WLAN-Verbindungen, unabhängig von den einzelnen Funktionseinstellungen.

:::

Die funktionsspezifischen Einstellungen greifen nur, wenn **Mobilen Datenzugriff erlauben** aktiviert ist.

So können Sie genau festlegen, welche OpenCloud-Funktionen Ihre mobile Datenverbindung verwenden dürfen, und unnötigen Datenverbrauch vermeiden.
