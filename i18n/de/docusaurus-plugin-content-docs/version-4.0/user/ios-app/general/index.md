# Allgemein

Erfahren Sie, wie Sie die OpenCloud iOS App installieren und einrichten, sich mit der Benutzeroberfläche vertraut machen und sie nach Ihren Wünschen konfigurieren.

## In diesem Abschnitt

- [Installieren der iOS App](./installation.md)  
  Laden Sie die OpenCloud iOS App aus dem App Store herunter und installieren Sie sie.

- [Einrichten der OpenCloud iOS App](./set-up.md)  
  Verbinden Sie die App nach der Installation mit Ihrem OpenCloud-Konto.

- [Übersicht](./overview/)  
  Lernen Sie das Konto-Menü und das Dateilisten-Menü kennen.

- [Einstellungen](./settings/)  
  Konfigurieren Sie die Benutzeroberfläche, die Datennutzung und weitere erweiterte Optionen.
