# Erstellen - Umbenennen - Verschieben

Sie können ganz einfach neue Text-, Tabellen- und Präsentationsdateien in der Weboberfläche erstellen. Diese können Sie auch umbenennen oder an einen anderen Speicherort verschieben

## Erstellen von Dateien und Ordnern

- Klicken Sie auf die Schaltfläche "+ Neu" oben in der Actionbar.
  <img src={require("./img/create-rename-move/new-button.png").default} width="1920"/>
- Wählen Sie im Menü den gewünschten "Dateityp", den Sie erstellen möchten.
  <img src={require("./img/create-rename-move/create-options.png").default} width="400"/>
- Geben Sie einen Namen für die Datei oder den Ordner in das angezeigte Fenster ein.
  <img src={require("./img/create-rename-move/create-new-folder.png").default} width="400"/>
- Klicken Sie auf "Erstellen".
  <img src={require("./img/create-rename-move/create-button.png").default} width="400"/>

  Fertig! Die Datei oder der Ordner wurde erstellt.
  Wenn es sich um eine Datei handelt, wird sie automatisch im Bearbeitungsmodus geöffnet.

## Umbenennen von Dateien und Ordnern

- Öffnen Sie das Kontextmenü, indem Sie entweder: - mit der rechten Maustaste auf die Datei oder den Ordner klicken, - oder indem Sie auf die drei Punkte rechts in der entsprechenden Zeile klicken.
  <img src={require("./img/create-rename-move/three-dot-menu.png").default} width="1920"/>
- Wählen Sie die Option „Umbenennen“.

  <img src={require("./img/create-rename-move/rename-option.png").default} width="400"/>

- Geben Sie den neuen Namen für die Datei oder den Ordner in das angezeigte Fenster ein. Bestätigen Sie, indem Sie auf "Umbenennen" klicken.
  <img src={require("./img/create-rename-move/rename-popup.png").default} width="400"/>
  <img src={require("./img/create-rename-move/rename-button.png").default} width="400"/>

Fertig! Der neue Name wird angewendet.

## Verschieben von Dateien und Ordnern

- Treffen Sie eine Auswahl:
  - Wählen Sie die Datei(en) oder den/die Ordner aus, die Sie verschieben möchten.
    <img src={require("./img/create-rename-move/mark-folder.png").default} alt="Dateien oder Ordner auswählen" width="1920"/>

- Öffnen Sie das Kontextmenü:
  - Klicken Sie mit der rechten Maustaste auf die Datei oder den Ordner.
  - Alternativ klicken Sie auf die drei Punkte in der entsprechenden Zeile.
    <img src={require("./img/create-rename-move/three-dot-menu.png").default} alt="Kontextmenü" width="1920"/>

- Wählen Sie die Option **„Ausschneiden“** aus dem Menü.
  <img src={require("./img/create-rename-move/right-click-cut-option.png").default} alt="Option „Ausschneiden“ im Kontextmenü" width="400"/>

  Alternativ können Sie die Funktion **„Ausschneiden“** in der Aktionsleiste am oberen Rand verwenden.

  <img src={require("./img/create-rename-move/actionbar-cut-option.png").default} alt="Option „Ausschneiden“ in der Aktionsleiste" width="1920"/>

- Ziel auswählen:
  - Navigieren Sie zu dem Ordner oder Speicherort, an den die Datei oder der Ordner verschoben werden soll.
  - Klicken Sie in der Aktionsleiste auf **„Hier einfügen“**.
    <img src={require("./img/create-rename-move/paste-here-button.png").default} alt="Schaltfläche „Hier einfügen“" width="1920"/>

  <img src={require("./img/create-rename-move/pasted-file.png").default} alt="Verschobene Datei" width="1920"/>

Die Datei oder der Ordner wurde nun an den neuen Speicherort verschoben und befindet sich nicht mehr am ursprünglichen Speicherort.
