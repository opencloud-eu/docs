# Übersicht

Die OpenCloud Android-App bietet eine intuitive Benutzeroberfläche, mit der Sie einfach auf Ihre Dateien zugreifen, Ihr Konto verwalten und zwischen den wichtigsten Funktionen der App navigieren können. Dieser Abschnitt stellt die wichtigsten Elemente der Benutzeroberfläche vor und erklärt, wie Sie diese im täglichen Einsatz effizient nutzen.

## In diesem Abschnitt

- [Kontomenü](./account-overview.md)  
  Lernen Sie das Kontomenü kennen, greifen Sie auf Ihre Kontoeinstellungen zu, zeigen Sie Ihren Speicherverbrauch an und finden Sie hilfreiche Ressourcen wie die Dokumentation und die Datenschutzerklärung.

- [Dateilisten-Menü](./account-fileslist.md)  
  Lernen Sie die Hauptansicht Ihrer Dateien kennen, erfahren Sie mehr über die Navigation sowie die Sortier- und Ansichtseinstellungen und greifen Sie schnell auf persönliche Dateien, Freigaben, Bereiche, Uploads und Offline-Inhalte zu.
