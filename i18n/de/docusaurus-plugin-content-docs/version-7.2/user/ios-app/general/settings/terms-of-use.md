# Nutzungsbedingungen

Der Bereich **Nutzungsbedingungen** bietet schnellen Zugriff auf die OpenCloud-Nutzungsbedingungen im Standardbrowser Ihres Geräts.

## Nutzungsbedingungen öffnen

1. Öffnen Sie die **OpenCloud**-App.
2. Tippen Sie auf **Einstellungen**.
3. Tippen Sie im Bereich **Weitere** auf **Nutzungsbedingungen**.

Wenn Sie diese Option auswählen, wird ein Bestätigungsdialog angezeigt, bevor die App verlassen wird.

<img src={require("../../img/settings/terms-of-use/terms of use button.png").default}
alt="Schaltfläche Nutzungsbedingungen" height="650"/>

## Öffnen der Website bestätigen

Es erscheint ein Bestätigungsdialog mit der Frage, ob Sie die externe Website öffnen möchten.

- Tippen Sie auf **OK**, um fortzufahren.
- Tippen Sie auf **Abbrechen**, um in der OpenCloud-App zu bleiben.

<img src={require("../../img/settings/terms-of-use/pop up.png").default}
alt="Pop-up" height="650"/>

## Nutzungsbedingungen anzeigen

Nach der Bestätigung öffnet OpenCloud die Nutzungsbedingungen im Standardbrowser Ihres Geräts.

Die Seite enthält die vollständigen Lizenz- und Nutzungsbedingungen für die Verwendung der OpenCloud-Software und zugehöriger Komponenten.

<img src={require("../../img/settings/terms-of-use/terms of use page.png").default}
alt="OpenCloud-Nutzungsbedingungen" height="650"/>

:::tip

Für den Zugriff auf die Nutzungsbedingungen ist eine Internetverbindung erforderlich, da die Inhalte von der offiziellen OpenCloud-Website geladen werden.

:::

:::note

Beim Öffnen der Nutzungsbedingungen wird der Standardbrowser Ihres Geräts gestartet. Sie können jederzeit zur OpenCloud-App zurückkehren.

:::
