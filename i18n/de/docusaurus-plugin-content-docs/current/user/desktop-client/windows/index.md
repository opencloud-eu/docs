# OpenCloud Desktop Client für Windows

Der OpenCloud Desktop Client für Windows ermöglicht die Synchronisierung von Dateien zwischen Ihrem Windows-Gerät und Ihrer OpenCloud-Instanz. Dieser Abschnitt begleitet Sie durch den gesamten Lebenszyklus des Desktop Clients – von der Installation und Ersteinrichtung über die Konfiguration der Synchronisierung und der Anwendungseinstellungen bis hin zur Verwendung des Windows Virtual File System (VFS).

## In diesem Abschnitt

- [Installation](./installation.md)  
  Installieren Sie den OpenCloud Desktop Client aus dem Microsoft Store.

- [OpenCloud Desktop einrichten](./set-up.md)  
  Verbinden Sie Ihr OpenCloud-Konto und führen Sie die Ersteinrichtung des Desktop Clients durch.

- [Synchronisierungseinstellungen unter Windows](./sync-settings.md)  
  Erfahren Sie, wie das Windows Virtual File System (VFS) funktioniert, verwalten Sie die lokale Verfügbarkeit von Dateien und konfigurieren Sie das Synchronisierungsverhalten.

- [Einstellungen unter Windows](./settings.md)  
  Passen Sie den Desktop Client an, indem Sie allgemeine, erweiterte, Netzwerk- und Bandbreiteneinstellungen konfigurieren.
