# OpenCloud Desktop Client für Linux

Der OpenCloud Desktop Client für Linux ermöglicht die Synchronisierung von Dateien zwischen Ihrem Linux-System und Ihrer OpenCloud-Instanz. Dieser Abschnitt begleitet Sie durch den gesamten Lebenszyklus des Desktop Clients – von der Installation und Ersteinrichtung über die Konfiguration der Synchronisierung und der Anwendungseinstellungen bis hin zum Upgrade auf neuere Versionen.

## In diesem Abschnitt

- [Installation](./installation.md)  
  Installieren Sie den OpenCloud Desktop Client unter Linux mithilfe von AppImage und AppImageLauncher.

- [OpenCloud Desktop einrichten](./set-up.md)  
  Verbinden Sie Ihr OpenCloud-Konto und führen Sie die Ersteinrichtung des Desktop Clients durch.

- [Synchronisierungseinstellungen unter Linux](./sync-settings.md)  
  Konfigurieren Sie das Synchronisierungsverhalten, wählen Sie aus, welche Ordner synchronisiert werden sollen, und verwalten Sie bestehende Synchronisierungsverbindungen.

- [Einstellungen unter Linux](./settings.md)  
  Passen Sie den Desktop Client an, indem Sie allgemeine, erweiterte, Netzwerk- und Bandbreiteneinstellungen konfigurieren.

- [OpenCloud Desktop Client unter Linux aktualisieren](./upgrade.md)  
  Erfahren Sie, wie Sie den OpenCloud Desktop Client auf eine neuere Version aktualisieren, ohne Ihre vorhandene Konfiguration oder bereits synchronisierte Dateien zu verlieren.
