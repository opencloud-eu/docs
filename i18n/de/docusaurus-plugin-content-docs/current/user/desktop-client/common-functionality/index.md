# Gemeinsame Funktionen

Dieser Abschnitt beschreibt Funktionen, die auf allen unterstützten Plattformen des OpenCloud Desktop Clients verfügbar sind. Hier erfährst du, wie du mehrere Konten verwaltest, Protokolldateien für die Fehlerbehebung sammelst, Synchronisierungsstatus verstehst, Dateikonflikte behebst und die Regeln für kompatible Dateinamen beachtest.

## In diesem Abschnitt

- [Mehrere Konten einrichten](./multiple-accounts.md)  
  Verbinde und verwalte mehrere OpenCloud-Konten innerhalb des Desktop Clients.

- [Protokolldateien sammeln](./logging.md)  
  Aktiviere die Protokollierung, sammle Diagnoseinformationen und teile Protokolldateien zur Fehlerbehebung.

- [Status des Desktop Clients](./desktop-client-states.md)  
  Erfahre, was die Statussymbole des Desktop Clients bedeuten und wie du darauf reagieren kannst.

- [Dateikonflikte beheben](./conflict-files.md)  
  Erfahre, warum Dateikonflikte auftreten und wie du sie sicher behebst.

- [Regeln für Dateinamen](./file-names.md)  
  Erfahre mehr über Einschränkungen bei Dateinamen und die Kompatibilität zwischen verschiedenen Betriebssystemen.

- [Dateien, die nicht hochgeladen werden dürfen](./not-allowed-upload-files.md)  
  Erfahren Sie, was passiert, wenn der Desktop Client eine Datei nicht hochladen kann und wo Sie diese lokal wiederfinden.
