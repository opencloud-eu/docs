# Einem Freund empfehlen

Mit der Funktion **Einem Freund empfehlen** können Sie andere schnell dazu einladen, OpenCloud auszuprobieren. Die App erstellt automatisch eine vorbereitete Einladungsnachricht mit einem Download-Link, den Sie über Ihre bevorzugte Kommunikations-App teilen können.

## Einladung teilen

1. Öffnen Sie die **OpenCloud**-App.
2. Tippen Sie auf **Einstellungen**.
3. Tippen Sie im Bereich **Weitere** auf **Einem Freund empfehlen**.

Wenn Sie diese Option auswählen, öffnet sich das Teilen-Menü Ihres Geräts mit einer vorbereiteten Einladungsnachricht.

<img src={require("../../img/settings/recommend-to-a-friend/einem freund empfehlen button.png").default}
alt="Schaltfläche Einem Freund empfehlen"
height="650"/>

## Einladung versenden

Wählen Sie die App aus, über die Sie die Einladung teilen möchten, zum Beispiel **Mail**, **Nachrichten** oder einen anderen unterstützten Freigabedienst.

Die Einladungsnachricht enthält bereits:

- einen vorausgefüllten Betreff,
- eine kurze Einladungsnachricht,
- einen Download-Link zu OpenCloud.

Geben Sie einfach den Empfänger ein und senden Sie die Nachricht.

<img src={require("../../img/settings/recommend-to-a-friend/nachricht feld.png").default}
alt="Nachrichtenfeld"
height="650"/>

:::tip

Sie können die Einladung mit jeder App teilen, die das iOS-Teilen-Menü unterstützt, darunter Mail, Nachrichten, WhatsApp, Signal, Telegram und viele weitere.

:::

:::note

Der erzeugte Download-Link verweist immer auf die neueste offizielle Version der OpenCloud-Mobile-App, die für das Gerät des Empfängers verfügbar ist.

:::
