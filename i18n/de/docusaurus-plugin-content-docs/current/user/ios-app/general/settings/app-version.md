# App-Version

Der Bereich **App-Version** zeigt detaillierte Versionsinformationen zur installierten OpenCloud-App an, einschließlich App-Version, Build-Nummer und SDK-Version.

## App-Versionsinformationen kopieren

1. Öffnen Sie die **OpenCloud**-App.
2. Tippen Sie auf **Einstellungen**.
3. Scrollen Sie bis zum Ende der Seite.
4. Tippen Sie auf **App-Version**.

<img src={require("../../img/settings/app-version/app versions button.png").default}
alt="Schaltfläche App-Version" height="650"/>

## Versionsinformationen kopiert

Nachdem Sie auf **App-Version** getippt haben, werden die vollständigen Versionsinformationen automatisch in die Zwischenablage Ihres Geräts kopiert.

Am oberen Bildschirmrand erscheint eine Bestätigung, dass die Versionsinformationen erfolgreich kopiert wurden.

<img src={require("../../img/settings/app-version/app versions pop up.png").default}
alt="Bestätigung zum Kopieren der App-Version" height="650"/>

:::tip

Wenn Sie einen Fehler melden oder den Support kontaktieren möchten, tippen Sie vor dem Erstellen Ihrer Meldung auf **App-Version**. Anschließend können Sie die kopierten Versionsinformationen direkt in Ihre Supportanfrage einfügen.

:::

:::note

Die kopierten Informationen können je nach OpenCloud-Version die App-Version, Build-Nummer, das Veröffentlichungsdatum, die Commit-ID und die SDK-Version enthalten.

:::
