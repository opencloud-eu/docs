# Sucheinstellungen

Mit den **Sucheinstellungen** können Sie festlegen, welcher Suchbereich standardmäßig verwendet wird, wenn Sie in der OpenCloud iOS-App nach Dateien und Ordnern suchen.

## Sucheinstellungen öffnen

1. Öffnen Sie die **OpenCloud**-App.
2. Tippen Sie auf **Einstellungen**.
3. Tippen Sie im Bereich **Benutzeroberfläche** auf **Sucheinstellungen**.

Auf der Seite **Sucheinstellungen** können Sie den Standardsuchbereich für neue Suchvorgänge auswählen.

<img src={require("../../img/settings/search-settings/sucheinstellungen button.png").default} alt="Schaltfläche Sucheinstellungen" height="650"/>

## Verfügbare Suchbereiche

Sie können einen der folgenden Standardsuchbereiche auswählen:

- **Ordner** – Durchsucht nur den aktuell geöffneten Ordner.
- **Ordnerbaum** – Durchsucht den aktuell geöffneten Ordner und alle darin enthaltenen Unterordner.
- **Space** – Durchsucht nur den aktuell geöffneten Space.
- **Konto** – Durchsucht das gesamte persönliche Konto über alle Spaces hinweg.
- **Server** – Führt eine serverweite Suche aus.

<img src={require("../../img/settings/search-settings/sucheinstellungen menü.png").default} alt="Menü Sucheinstellungen" height="650"/>

## Funktionsweise der Sucheinstellungen

Der ausgewählte Suchbereich wird automatisch als Standard verwendet, wenn Sie eine neue Suche starten.

Sie können den Suchbereich während einer Suche weiterhin manuell ändern, ohne Ihre Standardeinstellung zu ändern.

:::tip

Wählen Sie den Suchbereich, der am besten zu Ihrer Arbeitsweise passt:

- **Ordner** für schnelle Suchvorgänge im aktuellen Verzeichnis.
- **Ordnerbaum**, wenn auch Unterordner einbezogen werden sollen.
- **Space**, um alle Inhalte innerhalb des aktuellen Spaces zu durchsuchen.
- **Konto**, um alle Dateien zu durchsuchen, auf die Sie Zugriff haben.
- **Server**, wenn Sie die breiteste Suche über den gesamten Server verwenden möchten.

:::

:::note

Eine Änderung des Standardsuchbereichs wirkt sich nur auf **zukünftige Suchvorgänge** aus. Bereits angezeigte Suchergebnisse werden nicht verändert.

:::
