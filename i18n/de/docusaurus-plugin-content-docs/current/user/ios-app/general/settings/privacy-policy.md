# Datenschutzerklärung

Der Bereich **Datenschutzerklärung** bietet schnellen Zugriff auf die offizielle Datenschutzerklärung von OpenCloud im Standardbrowser Ihres Geräts.

## Datenschutzerklärung öffnen

1. Öffnen Sie die **OpenCloud**-App.
2. Tippen Sie auf **Einstellungen**.
3. Tippen Sie im Bereich **Weitere** auf **Datenschutzerklärung**.

Wenn Sie diese Option auswählen, wird ein Bestätigungsdialog angezeigt, bevor die App verlassen wird.

<img src={require("../../img/settings/privacy-policy/datenschutzrichtlinie button.png").default}
alt="Schaltfläche Datenschutzerklärung" height="650"/>

## Öffnen der Website bestätigen

Es erscheint ein Bestätigungsdialog mit der Frage, ob Sie die externe Website öffnen möchten.

- Tippen Sie auf **OK**, um fortzufahren.
- Tippen Sie auf **Abbrechen**, um in der OpenCloud-App zu bleiben.

<img src={require("../../img/settings/privacy-policy/pop up.png").default}
alt="Pop-up" height="650"/>

## Datenschutzerklärung anzeigen

Nach der Bestätigung öffnet OpenCloud die offizielle Datenschutzerklärung im Standardbrowser Ihres Geräts.

Die Seite enthält Informationen zu:

- Datenverarbeitung
- Datenschutzpraktiken
- DSGVO-Konformität
- Ihren Rechten in Bezug auf personenbezogene Daten

<img src={require("../../img/settings/privacy-policy/datenschutzrichtlinie seite.png").default}
alt="OpenCloud-Datenschutzerklärung" height="650"/>

:::tip

Für den Zugriff auf die Datenschutzerklärung ist eine Internetverbindung erforderlich, da sie von der offiziellen OpenCloud-Website geladen wird.

:::

:::note

Beim Öffnen der Datenschutzerklärung wird der Standardbrowser Ihres Geräts gestartet. Sie können jederzeit zur OpenCloud-App zurückkehren.

:::
