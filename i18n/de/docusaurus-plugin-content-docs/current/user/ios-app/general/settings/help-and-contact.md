# Hilfe & Kontakt

Der Bereich **Hilfe & Kontakt** bietet schnellen Zugriff auf OpenCloud-Ressourcen, darunter die offizielle Dokumentation, der GitHub-Issue-Tracker und die Community-Diskussionen.

## Hilfe & Kontakt öffnen

1. Öffnen Sie die **OpenCloud**-App.
2. Tippen Sie auf **Einstellungen**.
3. Scrollen Sie zum Bereich **Weitere**.
4. Tippen Sie auf **Hilfe & Kontakt**.

Auf der Seite **Hilfe & Kontakt** erhalten Sie Zugriff auf Dokumentation und Community-Support-Ressourcen.

## Verfügbare Ressourcen

Die Seite **Hilfe & Kontakt** bietet die folgenden Ressourcen:

- **Dokumentation anzeigen** – Öffnet die offizielle OpenCloud-Dokumentation mit Benutzerhandbüchern, Einrichtungsanleitungen und Funktionsbeschreibungen.
- **Problem melden** – Öffnet den GitHub-Issue-Tracker, in dem Sie Fehler melden oder neue Funktionen vorschlagen können.
- **GitHub Discussions besuchen** – Öffnet die OpenCloud-Community-Diskussionen, in denen Sie Fragen stellen, Ideen teilen und Hilfe von anderen Nutzern erhalten können.

Verwenden Sie diese Ressourcen, wenn Sie weitere Informationen benötigen, ein Problem melden möchten oder mit der OpenCloud-Community interagieren möchten.

## Wann welche Ressource verwenden?

| Ressource                  | Empfohlen für                                                              |
| -------------------------- | -------------------------------------------------------------------------- |
| **Dokumentation anzeigen** | Informationen zu OpenCloud-Funktionen und Schritt-für-Schritt-Anleitungen. |
| **Problem melden**         | Fehler, unerwartetes Verhalten oder Vorschläge für neue Funktionen.        |
| **GitHub Discussions**     | Fragen, Community-Support und Ideenaustausch.                              |

:::tip

Wenn ein Problem auftritt, sollten Sie zuerst die Dokumentation prüfen. Wenn das Problem weiterhin besteht, suchen Sie in GitHub Discussions oder erstellen Sie einen GitHub-Issue, falls Sie einen Fehler gefunden haben.

:::
