# Freigaben

Die OpenCloud Android-App ermöglicht es Ihnen, Dateien und Ordner sicher mit anderen Benutzern, Gruppen oder über öffentliche Links zu teilen. Dieser Abschnitt erklärt die verfügbaren Freigabemethoden und zeigt Ihnen, wie Sie Berechtigungen konfigurieren und Inhalte direkt von Ihrem Android-Gerät freigeben.

## In diesem Abschnitt

- [Per Einladung freigeben](./invite.md)  
  Geben Sie Dateien und Ordner für Benutzer und Gruppen innerhalb Ihrer OpenCloud-Instanz frei und konfigurieren Sie deren Zugriffsberechtigungen.

- [Per Link freigeben](./links.md)  
  Erstellen Sie passwortgeschützte öffentliche Links, konfigurieren Sie Linkeinstellungen wie Ablaufdaten und teilen Sie Dateien und Ordner sicher mit anderen.
